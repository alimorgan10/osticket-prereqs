<h2>Installation Steps</h2>

<p>
<img src="https://i.imgur.com/chyaFyl.png" height="80%" width="80%" alt="Disk Sanitization Steps"/>
</p>
<p>
Step one: make a resource group in which to put a VM in. Visit https://portal.azure.com/#home and input "Resouce group" into the search bar, then select the appropriate option. Name the group "RG-osticket" and make note the region you select. *The region should be the same for the Resource group and the VM. To create the VM go to https://portal.azure.com/#home and input "Virtual Machine" into the search bar, then select the appropriate option. When creating the VM select the resourse group just created, "RG-osticket". Name the VM, "VM-osticket". *Select the same region choosen for the RG-osticket. No redundancy is required for the VM and the Security can be set to "Standard". The image will be Windows 10 Pro. The size of the VM needs to contain at least two vcpus. *Make note of the username and password chosen for the administrator account. Inbound ports will be allowed and set to RDP 3389. Once the licensing is agreed to all other options (Disks, Networking, Managment, Advanced, and Tags) can be left with the autopopulated settings, just click "Review and Create" to deploy the VM.
</p>
